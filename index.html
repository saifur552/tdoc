HTML Basics

HTML (Hypertext Markup Language) is the standard markup language for creating web pages.

HTML elements are represented by tags, for example:

<p>This is a paragraph.</p>

Basic structure of an HTML document:

<!DOCTYPE html> <html> <head></head> <body></body>
  • HTML Tags
  • Attributes

CSS Basics

CSS (Cascading Style Sheets) is used to style and layout web pages.

It allows you to apply styles to HTML elements, such as setting colors, fonts, and layout:

body { background-color: lightblue; }

You can also use external style sheets:

<link rel="stylesheet" href="styles.css">
  • Selectors
  • Properties
  • Responsive Design
JavaScript Overview

JavaScript is a programming language used to create interactive effects on web pages.

You can use JavaScript to manipulate HTML and CSS, and even handle events:

document.getElementById('main-doc').style.color = 'blue';

JavaScript can also interact with web servers using APIs.

  • Variables and Data Types
  • Functions
  • DOM Manipulation
APIs and Fetch

APIs (Application Programming Interfaces) allow your web pages to interact with external services.

You can use the Fetch API to retrieve data from a server:

fetch('https://api.example.com/data')

APIs can return data in formats such as JSON or XML, which you can use in your web app.
